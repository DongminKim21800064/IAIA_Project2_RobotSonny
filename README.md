## Introduction

이 Respository는 한동대학교 2023년 1학기에 진행된 Industrial AI & Automation 수업의 기말 프로젝트인, **Robot Sonny (Airhockey Robot)** 에 대한  **Tutorial**  이 포함되어 있습니다.

Automation 프로젝트는 Indy-10(Neuromeka)의 6축 로봇팔을 제어하여 특정 임무를 수행하는 시나리오를 구성하며 로봇팔 제어 경험을 학습하기 위함과 저학년 학생들에게 로봇 체험을 통해 긍정적인 경험을 선사하여 기계제어공학부의 홍보를 목적으로 한다.
